# デプロイ前チェックリスト

デプロイを実行する前に、以下の項目を確認してください。

## サーバー情報の確認

- [ ] SSH接続情報（ユーザー名、ホスト名）を確認
- [ ] サーバー上のプロジェクトディレクトリのパスを確認
- [ ] Webサーバーのドキュメントルートのパスを確認

## データベース情報の確認

- [ ] データベース名
- [ ] データベースユーザー名
- [ ] データベースパスワード
- [ ] データベースホスト（通常は `localhost`）

## 初回デプロイ時の確認事項

- [ ] サーバーにGitがインストールされているか
- [ ] サーバーにComposerがインストールされているか
- [ ] サーバーのPHPバージョンが8.2以上か
- [ ] データベースが作成されているか
- [ ] データベースユーザーに適切な権限が付与されているか

## デプロイ前のローカル確認

- [ ] ローカルで動作確認済みか
- [ ] Gitに最新の変更をコミット・プッシュ済みか
- [ ] エラーや警告がないか

## デプロイ後の確認事項

- [ ] http://app-portfolio.xvps.jp/shiori にアクセスできるか
- [ ] ログインページが表示されるか
- [ ] データベース接続が正常に動作するか
- [ ] エラーログに問題がないか

## トラブル時の確認

### データベース接続エラー
```bash
# サーバー上で確認
mysql -u DBユーザー名 -p -e "SHOW DATABASES;"
```

### ファイルパーミッションエラー
```bash
# サーバー上で確認
ls -la php/public/
```

### PHPエラー
```bash
# サーバー上で確認
tail -f /var/log/apache2/error.log
# または
tail -f /var/log/php_errors.log
```
